# fullstack_developer_capstone

## Project Description
This is a Full Stack Application for a Car Dealership, developed as part of the IBM Full Stack Software Developer Capstone Project. It uses Django for the backend and React for the frontend.

## Features
- User Authentication (Login, Logout, Registration)
- View List of Car Dealerships
- View Dealer Reviews and Details
- Add New Reviews for Dealerships
- Sentiment Analysis for User Reviews
